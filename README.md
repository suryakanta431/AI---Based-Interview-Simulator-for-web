# 🤖 AI-Powered 3D Interview Simulator

An immersive AI-powered interview simulation platform built with **Unity WebGL**, featuring real-time **voice-to-voice interaction**, intelligent AI interviewers, and interactive 3D environments designed to simulate realistic interview experiences.
  


---

# 🎮 Project Overview

This project is designed to create a realistic virtual interview environment where users can interact with AI-powered interviewers using natural voice communication.

The simulator focuses on:
- Real-time voice conversations
- AI-driven interview interactions
- Immersive 3D interview environments
- Web-based accessibility using Unity WebGL
- Interactive learning and practice experiences
